# Zakah Calculator

A simple web application for calculating Zakah.

## About

This project provides an easy-to-use interface for calculating Zakah based on various assets and liabilities.

## Features

- **Automatic Gold Price Integration**: Automatically fetches the current gold price in your local currency to calculate the Nisab for cash and trade assets.
- **Currency Support**: Calculates Zakat in multiple currencies (SAR, USD, AED, EGP, etc.).
- **Zakat Types**: Supports calculations for Gold, Silver, Cash, Trade Assets, and Agriculture.
- **AI Integration**: Ask questions about Zakat rulings powered by Gemini.

## Usage

Simply open the index.html file in your browser to use the calculator.
